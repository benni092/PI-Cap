# PI-Cap
PI-Cap is a collection of scripts for video-recording on a Raspberry-Pi. It has mainly tapeless cameras in mind.
You can build as an example a opensource powerplay!

# PI-Cap AIO
This is the latest PI-Cap image ready to burn to sd-card.
It has many different record scripts included. Just copy the script you like from the Pi-Cap folder (located in the root of the boot partition) next to config.txt and it will automatically executed.
